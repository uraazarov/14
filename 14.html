Каскадные таблицы стилей

При оформлении HTML-документов приходиться по несколько раз прописывать однотипные теги.Использование каскадных таблиц стилей (CSS) позволяет значительно упростить эту работу.

Поддержка стандарта кскадных таблиц стилей и их программируемых элементов появилась начиная с версии 4.0 и Netscape, и Internet Explorer.

Существует три основных варианта использования таблиц стилей.

Вариант первый,описание стиля распологается непосредственно внутри тега элемента, который вы описывате. Это делается с помощью параметра STYLE. Этот метод нежелателен и понятно почему:он приводит к потере одного из основных преимущест CSS
